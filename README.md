文字冒險 RPG 小遊戲

這是一個以 C++ 撰寫的純文字（命令列）角色扮演遊戲，玩家需要透過冒險戰鬥、強化裝備、學習技能與抽獎來提升角色實力，最終目標是擊敗魔王、通關遊戲。


本專案為大一時期練習所寫，主要用於熟悉 C++ 的基本語法、迴圈與分支結構、陣列操作以及簡單的遊戲狀態管理。



遊戲特色


回合制戰鬥：與隨機出現的魔物戰鬥，可選擇攻擊、施放技能或逃跑。
裝備強化：花費金幣強化目前持有的武器，提升攻擊力。
技能系統：達到等級門檻後可學習技能（治癒之風、三重打擊、鋼鐵之盾、毒蛇打擊），戰鬥中可消耗魔力使用。
抽獎系統：花費金幣抽取武器，等級越高可抽到的武器稀有度越高，亦提供十連抽。
裝備切換：可在已擁有的武器中自由切換裝備。
每日精力與休息：每天有限的精力值（5 點），行動會消耗精力，精力耗盡需要休息以進入下一天。
等級成長：累積經驗值提升等級，等級提升會增加攻擊力、最大血量與魔力。


遊戲流程


開場動畫介紹世界觀。
每天開始時顯示天數，並重置精力值。
玩家可從以下選項中選擇行動：

冒險（觸發隨機戰鬥或發現寶藏）
強化裝備
學習技能
抽獎
裝備（切換武器）
休息（結束當天，進入下一天）



戰鬥中可選擇攻擊、使用技能或逃跑，擊敗魔物可獲得金幣與經驗值。
等級達到 10 時可挑戰魔王（怪物編號第 10 隻），擊敗後即通關遊戲。


開發環境


語言：C++
平台：Windows（使用 windows.h 的 Sleep() 與 conio.h 的 getch()，因此僅能在 Windows 環境下編譯執行）
編譯方式（以 g++ 為例）：


bashg++ -o game main.cpp
game.exe

操作方式

遊戲全程以鍵盤數字鍵輸入選擇，按下對應數字即可執行該行動，無需按下 Enter（部分流程除外）。
